コンテンツ(4) jQueryとHTML5

あらまし

コンテンツに関して、今回は、HTML5やjQueryを使ったリッチなコンテンツについて扱う。 前回用いたjQuery Mobileも実はHTML5とjQueryを使ったJavaScriptライブラリであり、HTML5とjQueryは、現在のモバイル対応を含めたリッチなWebコンテンツの大きな2つの根源といってよいだろう。 また、Googleドライブ上のワープロや表計算に代表されるような、Webブラウザ上で動作するアプリケーションの基盤でもある。HTML5とjQuery、それらをも用いたライブラリに関して紹介するとともに、実際にこれらを用いた簡単なコンテンツの作成の実習を行う。

解説

HTML5

HTML5は、HTML規格の大幅な更新であり、これまでブラウザ上で直接は扱えなかったグラフィックスや動画等が扱えるようになりWebコンテンツに大きな変革をもたらしたといえよう。 その影響は、そのうちJava appletやFlashがなくなるかもしれない勢いである。HTML5はHTML自体の規格なので、Java appletやFlashのようなプラグインが必要ない。さらに、Java appletの利用が減ってきつつあり、また、90%以上の普及率を誇っていたFlashさえも事情が変わってきた。 アップルの公式サイト上のジョブズによるFlashについての見解 (2010年4月)にあるように、AppleはiOSでFlashの採用をやめた。経緯な詳細は、 Tech Crunchによる一連の経緯の解説 Ryan Lawler「ジョブズの誇り：AppleがFlashとの戦いに勝つまで」(2010年4月) にある。 Flashの開発元のAdobeさえも 同社のWebサイトでの発表(2012年6月)で、Android 4.1以降でのFlash Playerのサポートを中止するとしている。

jQueryとそれを利用したJavaScriptライブラリ

jQueryは、Ajax、イベント処理、 動的なHTMLの書き換え等をサポートするJacaScriptライブラリである。 また、その機能を拡張する多くのJavaScriptライブラリ(jQueryプラグイン)が開発されており、Webアプリケーションのブラウザ(クライアント)側での処理能力を飛躍的に進展させるとともに、Webコンテンツに関しても今まではアプリケーションでしかできなかったような様々なリッチな機能を入れ込むことが可能になってきた。

jQueryのプラグインは、jQueryのホームページのトップのタブにあるように、内部で同時に開発されている jQuery UIや jQuery Mobile 以外にも、jQueryの公式サイトにあるプラグイン登録サイト The jQuery Plugin Registry に多数が登録されている(2014年5月時点で1,000以上登録あり)。 ここでキーワードで検索することができ、例えば、「グラフのプロット」 等で検索することができる。 さらに、このサイトに登録されていないものでも、よく使われているものが多数ある。 次節の「実例」で、HTML5やjQueryを用いたライブラリをいくつか紹介する。

HTML5やjQueryの他の重要な側面

今回は、HTML5やjQueryのコンテンツへの活用としての側面に関して扱った。 しかし、今回扱っていない内容として、

  • HTML5の位置情報機能の Geolocation APIを用いると、モバイルデバイス等の現在位置等を取得できる。
  • HTML5の動画再生機能のhttp://www.htmq.com/html5/004.shtml を用いると、Webブラウザだけで動画の再生が可能になる。ただし、再生可能な動画形式が限られているので注意が必要である。
  • HTML5のカメラ制御機能の Media Capture and StreamsやMedia Capture and Streamsを用いると、ブラウザからビデオ撮影やストリーミング等が行えるようになる。
  • HTML5のでビデオ/オーディオの通信や、データ通信を行うための規格 WebRTCを用いると、ブラウザ間のテレビ会議等が可能になる。
  • jQueryの 非同期通信(Ajax)と動的Webページ内容の書き換え機能を用いると、ユーザにページのロードや通信が終わるまで待たせることがなく、現在表示ページを順次書き換えていくことができる。
  • jQueryの JSONPデータの通信機能( REST形式の通信)を用いると、複数のサーバと同時に通信することが容易に実現でき、 マッシュアップが実現できる。 例えば、マッシュアップのサンプルプログラム では、外部の２つのサーバのJSONPによるRFSTサービス(Web API)を利用して、熊本県の天気予報と日の出日の入りの時刻を表示している。
等々あり、Webに改革を起こしつつある。

実例

HTML5とjQueryを使ったJavaScriptライブラリ

これはほんの一部であるが、HTML5とjQueryを使ったJavaScriptライブラリを紹介する。 他によい例を見つけたら、タスク提出時に情報共有のため紹介してほしい。 第8回で紹介したjQuery Mobileも代表的なものであるが、ここでは省略するので、第8回を参照してほしい。

  • jQuery UI : jQuery UI は、ボタンやチェックボックス等の様々なウィジェットや、ドラッグやリサイズといったマウスによる対話性、アニメーション効果等のエフェクトを与える。 これらは、Webアプリケーションの利便性の向上だけでなく、Webコンテンツの可能性も拡げる。
  • jsTree : jsTreeは、ファイルやフォルダー構造の表示等に使われるツリー表示を行うためのライブラリです。 デモ にあるように、様々なリスト表示が可能である。 ただ表示するだけではなく、本当に開いたり閉じたりできる。
  • DataTables : DataTablesは、表(table)に関する非常に多機能なライブラリで、デモ にあるように、綺麗な表が作成できるだけではなく、表示行数の変更、カラム毎の並べ替え、検索(抽出)等もできる。つまり、アプリケーションでしかできなかったようなことが可能になる。
  • Flot Examples : HTML5とjQueryを使ってグラフを描くライブラリ(サンプルページ)。対話性を重視。
    Selection and zooming の例の用に、選択範囲やズームもできます。これはイメージの貼り付けでは絶対にできないことで、まるでアプリケーションのようである。
  • Chart.jsもグラフをプロットするシンプルなライブラリ。
  • impress.js | presentation tool based on HTML5 & CSS3 : HTML5,CSS3を用いたプレゼンページ作成用ライブラリ(デモページ)。
    HTML5のグラフィック能力の高さが実感できる。
    第159回ランチョンセミナー でも紹介した。
  • HTML5 のデモ | Demo Studio | MDN : Mozilla Developer Networkで集められているもの。一般的なもので多数あり。
  • HTML5 Demos & Samples : Demoサイト集。
