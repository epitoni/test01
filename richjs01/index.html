HTML5 and its enhanced JavaScript libraries / HTML5とそれによって強化されたJavaScriptライブラリ
This content is ready for mobile device by Bootstrap.

The goal is to understand what has changed with the advent of HTML5 and the many JavaScript libraries that have been developed to take advantage of its functionality, and to be able to practically implement them.
-
HTML5の出現でなにが変わったか，またその機能を便利に活用するために開発された多くのJavaScriptライブラリがあることについて理解し， 実際に利用できるようになることを目指す．

  • HTML5, an HTML standard that supports multimedia such as graphics, WebAPI, a mechanism for APIs on the Web, and JavaScript libraries that simplify usage or realizing higher functions are the major sources of rich Web applications such as SPA. In other words, it is the underlying technology for applications that run in a web browser, such as word processing and spreadsheets on Google Drive. In this section, we will introduce various JavaScript libraries developed based on HTML5 and jQuery, which are indispensable for developing rich applications on the Web.
    -
    グラフィックス等マルチメディアをサポートしたHTMLの規格 HTML5、 Web上でのAPIの仕組み WebAPI、 それらの操作を簡単にしたり高い機能を実現した各種ライブラリは、 SPA 等のリッチなWebアプリケーションの大きな源流といってよいだろう。 すなわち、Googleドライブ上のワープロや表計算に代表されるような、Webブラウザ上で動作するアプリケーションの基盤技術である。 ここでは、HTML5の基に開発された様々なJavaScriptライブラリを紹介するが、 それらはWeb上のリッチなアプリケーション開発になくてはならないものである。

  • HTML5 is a major update of the HTML standard, and has revolutionized Web content by enabling the handling of graphics, video, etc. that could not be handled directly on a Web browser before HTML5. Before HTML5, it was common to use Adobe's FLASH plug-in to handle graphics. Thanks to the contributions of Steve Jobs and others, this is no longer necessary. For more details, see Steve Would Be Proud: How Apple Won The War Against Flash (2012-06-30), etc. Adobe, the developer of Flash, announced (in July 2017) that it will stop updating and distributing Flash at the end of 2020.
    -
    HTML5は、HTML規格の大幅な更新であり，それまでWebブラウザ上で直接は扱えなかったグラフィックスや動画等が 扱えるようになりWebコンテンツに大きな変革をもたらしたといえよう． HTML5以前は，Adobe社のFLASHプラグインを入れることでグラフィックス等を扱うのが一般的であった． スティーブジョブス等の貢献もあり，現在はそのような必要がなくなった． 経緯な詳細は， 「ジョブズの誇り：AppleがFlashとの戦いに勝つまで」(2012年6月)等にある． Flashの開発元のAdobeは，2020年の年末をもってFlashのアップデート及び配布を終了する(2017年7月) と発表した．

  • jQuery has been recognized as the de facto standard library for asynchronous communication, DOM manipulation, and more. In fact, it is still widely used today. For example, it is listed as The Most Popular JavaScript Libraries in The 40 Best JavaScript Libraries and Frameworks for 2023.
    -
    jQueryは，非同期通信やDOM操作等．デファクトスタンダードなライブラリだと目されてきました． 実際現在も大変多く使われています． 例えば， The 40 Best JavaScript Libraries and Frameworks for 2023 で，The Most Popular JavaScript Librariesになっています．

  • However, with the arrival of HTML5 and the enhanced functionality of JavaScript itself, some are doubtful about its future ( The future of jQuery, jQuery Mobile is no longer supported, "Is jQuery dead in 2023?).
    -
    しかし，HTML5の出現とJavaScript自体の高機能化等から，将来を危ぶむ声もあります ( The future of jQuery, jQuery Mobile is no longer supported, 【2022年5月現在】jQueryはオワコン？勉強の必要性や将来性についてまとめてみた, jQuery UIとjQuery Mobileがついに開発終了, )．

  • Therefore, in this course, I started to remove jQuery from the contents and sample programs in FY2022. Work is continuing on this in FY2023, but please let me know if I have forgotten to remove it.
    -
    そこで，本講義では，2022年度から，内容やサンプルプログラムからjQueryを外し始めました． 2023年度も作業を継続していますが，消し忘れがあったらご連絡ください．

These are just a few examples of useful JavaScript libraries using HTML5. The library introduced here is so-called Vanilla.js, which does not depend on other JavaScript libraries (jQuery, etc.). jQuery, jQueryMobile, jQuery UI, jsTree, DataTables, and FlotExamples were removed from FY2023.
-
これらはほんの一例ですが，HTML5を使用した便利なJavaScriptライブラリを紹介します． ここで紹介するライブラリは，いわゆるVanilla.jsと呼ばれる他のJavaScriptライブラリ(jQuery等)に依存しないものです． 2023年度から，jQuery, jQueryMobile, jQuery UI, jsTree, DataTables, FlotExamplesを外しました．

  • Bootstrap:
    It supports Responsive web design and is useful for building a device-independent web front-end UI. It is often used to migrate from jQueryMobile, but may not be compatible with jQueryMobile (personal opinion).
    レスポンシブウェブデザイン に対応し、デバイスに関わらないWebのフロントエンドのUIの構築に役立つ． jQueryMobileからの移行に使われることも多いが，まり互換性はないかもしれません(個人的な見解)．
  • Grid.js:
    It is a very high-performance tabular library that makes it easy to create tables with search, sorting, paging, etc. (It depends on jQuery, but a well-known high-performance one called DataTables is available.)
    非常に高機能な表作成ライブラリで、検索、ソート、ページング等を備えた表が簡単に作成できます。 (jQueryに依存しますが，DataTablesという高機能なものが有名です)
  • Chart.js:
    A feature-rich, easy-to-use library for plotting a variety of graphs.
    様々なグラフをプロットするための高機能で使いやすいライブラリ．
  • D3.js:
    A powerful library that also handles SVG and excels at data visualization and animation. The examples show its wide range of applications.
    SVGも扱え、データの可視化やアニメーションが得意な強力なライブラリ。 例 を見ると、その応用範囲の広さがわかる。
  • TinyMCE:
    Add WYSIWYG editor functionality on web pages. It is built into many systems, such as Wordpress and Moodle, and shows that the editing was actually done on the browser side.
    Webページ上にWYSIWYGエディターの機能を追加する。 WordpressやMoodle等、多くのシステムに組み込まれていて、実はブラウザ側で編集していたことがわかる。

    Many demos. / デモ多数．
